# Proyecto Initializr – Plantilla web HTML5

Proyecto académico desarrollado como práctica de Lenguajes de Marcas.

## Descripción
Sitio web estático basado en una plantilla inicial generada con Initializr, cuyo objetivo es aprender la estructura base de un proyecto web moderno utilizando HTML5, CSS y JavaScript.

El proyecto sirve como punto de partida para el desarrollo de páginas web, aplicando buenas prácticas de organización y estructura del código.

## Funcionalidades
- Estructura HTML5 semántica (header, nav, article, section, aside, footer)
- Separación de contenido, estilos y scripts
- Uso de estilos base para normalización entre navegadores
- Inclusión de librerías JavaScript comunes

## Tecnologías utilizadas
- HTML5
- CSS
- JavaScript
- Normalize.css
- jQuery

## Contexto
Proyecto realizado durante el ciclo formativo de DAW como práctica del módulo de Lenguajes de Marcas.

## Qué he aprendido
- Estructuración correcta de páginas web con HTML5
- Organización de un proyecto web
- Uso de plantillas base para desarrollo frontend
